# Criando relatórios em PHP usando o Koolreport

## Requisitos

PHP 5.4, recomendado 7.4 ou superior

## Download

https://www.koolreport.com/getting-started#download

## Criar relatório

Descompactar e copiar para

mkdir /var/www/relatorio

Teremos

/koolreport
CarsByColor.php
CarsByColor.view.php
index.php
